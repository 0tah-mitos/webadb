基于 WebUSB API 实现，支持 Chrome / Edge 浏览器

🔑 无线配对（Android 11+）

在手机上：设置 → 开发者选项 → 无线调试 → 使用配对码配对设备
然后输入手机上显示的配对码和配对端口号

手机上显示的配对码

🔗 连接 ADB 设备

配对成功后，使用连接端口（不是配对端口）连接设备
在手机「无线调试」页面查看 IP 地址和端口
